Bubble Sort

Description

Bubble sort is a simple sorting algorithm. This sorting algorithm is comparison-based algorithm in which in each cycle each pair of adjacent elements is compared and the elements are swapped if they are not in order.
Reason Behind the Name Bubble Sort Because the effect is just like dropping of stone which leads to sparkling bubble at top Here Bubble represent the elements of data structure . The bigger bubbles reach the top faster than smaller bubbles.

Algorithm

We assume list is an array of n elements. We further assume that swap function swaps the values of the given array elements.

begin BubbleSort(list)
for all elements of list
if list[i] > list[i+1]
swap(list[i], list[i+1])
end if
end for
return list
end BubbleSort
IMPLEMENTATIONS
    
    void bubble_sort(int arr[], int n) {
    int c, d, t;
        
    for (c = 0 ; c < n - 1; c++) {
    for (d = 0 ; d < n - c - 1; d++) {
    if (arr[d] > arr[d+1]) {
            t = arr[d];
            arr[d] = arr[d+1];
            arr[d+1] = t;
          }
        }
        }
        }

Advantages:

  1. It is the simplest sorting approach.
  2. Best case complexity is of O(N) [for optimized approach] while the array is sorted.
  3. Using optimized approach, it can detect already sorted array in first pass with time complexity of O(N).
  4. Stable sort: does not change the relative order of elements with equal keys.
  5. In-Place sort.

Disadvantages:

  1. Bubble sort is comparatively slower algorithm.

Complexity

Best case O(n)
Average case O(n2)
Worst case O(n2)
Space complexity O(1)
